# 인사이트 로그 (Insights Log)

> **리서치 하네스의 주 주입 지점(intake).** 인터뷰·실험·관찰에서 배운 것이 들어오면
> **여기 맨 위에 한 항목**을 추가한다(최신순). 각 항목은 "무엇을 배웠나 → 무엇이 바뀌나"를 잇는다.
> 상세는 `research/`·`experiments/` 파일에, 가설 상태는 `HYPOTHESES.md`에, 제품/기술 결정은
> `DECISIONS.md`에 각각 남기고 여기서는 그것들을 `[[링크]]`로 연결만 한다.
>
> 항목 형식:
> ```
> ## YYYY-MM-DD · <제목>
> - **유입**: 무엇을 했고 무엇을 관찰했나 (출처: [[research/…]] 또는 [[experiments/…]])
> - **가설 영향**: H-XXX 미검증→흔들림 등
> - **결정/변경**: 제품·코드에 반영할 것 (있으면 [[DECISIONS]] 항목 링크)
> - **다음**: 후속 액션
> ```
